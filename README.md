# Breast Cancer Detection

A machine learning project to detect and classify breast cancer using a dataset of clinical features.

## Project Structure
* `data.csv` - The dataset containing patient clinical features and diagnostic labels.
* `Breast_Cancer_Detection.ipynb` - The Google Colab Notebook containing data preprocessing, model training, and evaluation[cite: 1].

## Getting Started

### Run on Google Colab
1. Open the `Breast_Cancer_Detection.ipynb` notebook file in Google Colab[cite: 1].
2. Upload `data.csv` directly to your Colab session runtime storage using the folder icon on the left panel[cite: 1].
3. Step through and execute the notebook cells[cite: 1].
